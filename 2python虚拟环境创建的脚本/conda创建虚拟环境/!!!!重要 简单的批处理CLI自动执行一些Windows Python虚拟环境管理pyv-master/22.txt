Python——venv虚拟环境（windows环境下）
创建虚拟环境
python -m venv 虚拟环境名（如：zzz）

进入目录
cd 虚拟环境名/Scripts (如下图)



激活虚拟环境
activate.bat



激活虚拟环境后可以pip install 跟项目用到的包，用 Python **.py 运行。如果有多个项目，就生成多个虚拟环境一一搭配。
好处：实现了项目环境隔离。
